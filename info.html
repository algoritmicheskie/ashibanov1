Советы новичкам

  1. Используй шапку, подвал, основной заголовок с помощью тегов:header,footer,main.
  2. Добавляй отступы с помощью тегов:margin,padding.
  3. Подбирай цвета по цветовому кругу.
  4. Соблюдай симметрию.
  5. Пиши код понятно и текст граммотно.
  6. Выделяй интерактивные элементы.
